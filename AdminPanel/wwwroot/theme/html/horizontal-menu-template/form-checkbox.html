      چک باکس های پایه

      چک باکس های سفارشی

      کلاس .custom-control .custom-checkbox را به عنوان یک دربرگیرنده اضافه کنید و .custom-control-label را برای دریافت خروجی بهتر اضافه کنید.

      چک باکس های سفارشی رنگی

      برای تغییر رنگ چک باکس های سفارشی کلاس .bg-{colorName} را به همراه .custom-control-input اضافه کنید.

      چک باکس های رنگی

      برای تغییر رنگ چک باکس از کلاس checkbox-{colorName} برای primary, secondary, success, danger, info, warning استفاده کنید.

      درخشش چک باکس ها

      از کلاس .checkbox-glow به همراه .checkbox.checkbox-{colorName} برای جلوه درخشش چک باکس ها استفاده کنید. رنگ درخشش بر اساس رنگ استفاده شده برای چک باکس خواهد بود.

      سایه چک باکس ها

      از کلاس .checkbox-shadow به همراه .checkbox برای جلوه سایه بر روی چک باکس ها استفاده کنید.

      چک باکس های دارای آیکن

      از .checkbox-icon به همراه .checkbox برای تغییر آیکن داخل چک باکس استفاده کنید.

      اندازه های چک باکس ها

      برای افزودن چک باکس با اندازه های مختلف، ما کلاس .checkbox-sm را برای چک باکس های کوچک داریم. آن را به همراه .checkbox اضافه کنید.

      سفارشی سازی کنید و به صورت زنده مشاهده کنید.
