      افزودن سطر ها

      سطر های جدید میتوانند به سادگی توسط متد API row.add() به DataTable اضافه شوند. به سادگی فانکشن API را با اطلاعاتی که برای سطر جدید باید استفاده شوند فراخوانی کنید (به صورت آرایه یا object). سطر های چندگانه میتوانند توسط متد rows.add() اضافه شوند (به جمع بودن rows توجه کنید). اطلاعات میتوانند توسط متد های row().data() و row().remove() به روز رسانی شوند.

      انتخابگرهای ستون با گزینه های برون بری و چاپ

      همه دکمه های برون بری اطلاعات یک گزینه exportOptions دارند که میتواند برای مشخص کردن اینکه چه اطلاعاتی و چگونه باید برون بری شوند استفاده شود. گزینه های داده شده به این پارامتر به صورت مستقیم به متد buttons.exportData() ارسال میشوند تا اطلاعات مورد نیاز دریافت شود.

      دکمه چاپ یک پنجره جدید در مرورگر کاربر باز خواهد کرد و به صورت خودکار عمل چاپ را انجام خواهد داد که به کاربر اجازه چاپ جدول را می دهد. پنجره پس از اتمام چاپ و یا لغو چاپ بسته خواهد شد.
